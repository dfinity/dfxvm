# dfxvm install

<MarkdownChipRow labels={["Reference"]} />

Installs a version of `dfx`. Can be used for official release versions and beta releases.

```bash
dfxvm install <version>
```

## Examples

```bash
dfxvm install 0.15.0
```

```bash
dfxvm install 0.15.2-beta.1
```
