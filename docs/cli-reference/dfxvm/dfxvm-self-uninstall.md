# dfxvm self uninstall

Uninstalls dfxvm and all versions of dfx.

## Usage

```bash
dfxvm self uninstall [--yes]
```

### Options

| Option                    | Description |
|---------------------------| --- |
| `--yes`                   | Skip confirmation prompt |

## Examples

Uninstall dfxvm and all versions of dfx, with an opportunity to confirm or cancel.

```bash
dfxvm self uninstall
```
