# dfxvm self uninstall

Uninstalls `dfxvm` and all versions of `dfx` with the opportunity to confirm or cancel.

```bash
dfxvm self uninstall [--yes]
```

### Options

| Option                    | Description |
|---------------------------| --- |
| `--yes`                   | Skip confirmation prompt |

## Examples

```bash
dfxvm self uninstall
```
