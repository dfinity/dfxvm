# dfxvm uninstall

<MarkdownChipRow labels={["Reference"]} />

Uninstalls a specific version of `dfx`. Does not uninstall `dfxvm` itself.

```bash
dfxvm uninstall <version>
```

## Example

```bash
dfxvm uninstall 0.15.1-beta.0
```
